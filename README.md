# SmartStock AI

Sistema de previsão de demanda utilizando Azure Machine Learning.

Projeto desenvolvido como parte do:

DIO - Microsoft Certification Challenge #5 - DP-100

## Objetivo

Prever vendas diárias para otimizar:

- estoque
- produção
- lucro
- tomada de decisão

## Stack

- Python
- Azure Machine Learning
- MLflow
- Scikit-learn
